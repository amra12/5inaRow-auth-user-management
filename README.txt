This is the project 5inaRow.

It is a Microservices project for the class Project Laboratory of BME.

HOW TO INSTALL NODE JS AND HOW TO RUN MY SERVICE: 

1- install node for mac , linux , windows , docker : https://nodejs.org/en/download/ 

2- install npm  : https://www.npmjs.com/get-npm

3-install the required dependencies in the directory of the project


npm install

Running the service:
````
node server.js

````
